## 功能特性

- **官方 API 支持**：基于腾讯官方开放 API 获取表情包资源。
- **命令行工具**：提供纯 CLI 交互，使用轻量且高效。
- **资源下载**：支持将表情包下载为 png 格式。

## 项目说明

本项目作为一个学习交流的工具 Demo，展示了如何调用官方接口获取特定资源，使用时请遵守相关规定并注意版权保护。
